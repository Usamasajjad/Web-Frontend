Basic Terminology
A grid has the following items:
1. Grid Columns: imagine vertical lines across the page. Use: grid-column-start/grid-column-end. 2. Grid Rows: imagine horizontal lines across the page. Use: grid-row-start/grid-row-end. 3. Grid Gaps: Used to set gaps between grid boxes. Use: gap/row-gap/column-gap. 4. Grid Lines: image the lines between columns and rows.
